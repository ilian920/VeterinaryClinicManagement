Технологиите заемат централно място в съвременния свят, позволявайки на обществото да създава по-безопасни, удобни и устойчиви условия за живот. Един от ключевите подходи за постигането на това е внедряването на интелигентни системи за мониторинг и анализ на параметри от околната среда. Измерването на температура и налягане е критично важно в множество повседневни приложения – от поддържането на оптимален микроклимат в домовете, до профилактиката и ранното установяване на потенциални технически дефекти в строителните конструкции и промишлените съоръжения.

С развитието на Интернет на нещата (IoT), възможностите за дистанционен мониторинг и контрол се разширяват значително. Съвременните IoT системи позволяват събирането, обработката и анализа на данни в реално време, като едновременно информират потребителя навременно или автоматично предприемат необходимите коригиращи действия.

Настоящата разработка представя комплексна IoT-базирана решение, комбинирана с мобилно приложение, предназначена за мониторинг и анализ на данни, събирани от множество сензори. Основният фокус е върху температурата и налягането, но функционалността е разширена чрез интегрирането на допълнителни сензорни модули:

Капацитивен сензор за отчитане на влажността на почвата
BH1750 – сензор за измерване на интензитета на светлината
MICS-5524 – сензор за оценяване на качеството на въздуха
AHT20 – комбиниран сензор за температура и относителна влажност
Целта на системата е да предостави всеобхватна и точна картина за текущото състояние на околната среда, като чрез това значително улесни хората в управлението на своята здравна и безопасност, както и в поддържането на оптимални условия за комфорт и производителност.

3.1.2 Софтуерни инструменти

1. ASP.NET Framework и платформа

ASP.NET Framework е мощна платформа за разработване на уеб приложения, разработена от Microsoft. Тя осигурява надеждна инфраструктура за изграждане на сложни бизнес приложения, включително поддръжка за безопасност, управление на състоянието и асинхронна обработка на данни. ASP.NET Framework позволява разработчиците да създават скалируемо и сигурно приложение, способно да обслужва множество едновременни потребители в ветеринарната клиника.

2. ASP.NET MVC (Model-View-Controller) архитектура

MVC е архитектурен модел, който разделя приложението на три взаимосвързани компонента:

Model – представя данните и бизнес логиката на приложението (пациентски информация, медицински записи, преглеждане на животи)
View – отговаря за презентацията на данните в потребителския интерфейс
Controller – управлява взаимодействието между Model и View, обработвайки потребителските заявки
Този подход позволява добра организация на кода, лесна поддръжка и тестване на приложението.

3. C# – езикът за програмиране

C# е съвременен, типизиран език за програмиране, разработен от Microsoft и работещ на платформата .NET Framework. Той поддържа обектно-ориентирано програмиране, обобщени типове (generics), асинхронно програмиране и други напредни функции. C# се използва за писането на цялата бизнес логика на системата за управление на ветеринарната клиника.

4. SQL Server / SQL Database – система за управление на бази от данни

Системата използва SQL Server или облачна SQL Database за съхранение на всички критични данни:

Информация за пациентите (животни)
Медицински истории и записи
Информация за собствениците
График на приемни часове
Данни за лекарства и материали
Счетоводни записи
SQL осигурява надеждно съхранение, бързо търсене и анализ на данните, необходими за ежедневното функционирането на клиниката.

5. Entity Framework – ORM (Object-Relational Mapping)

Entity Framework е ORM библиотека, която позволява на разработчиците да работят с бази данни, използвайки обекти на C# вместо директно SQL писане. Това упростява кода, намалява грешките и позволява лесна промяна на базата данни в бъдеще, без да се налага пренаписване на цялата логика.

7. Bootstrap – фреймворк за отзивчив дизайн

Bootstrap е фреймворк за CSS, който осигурява предварително проектирани компоненти (бутони, форми, таблици) и система за решетка. Той позволява бързото изграждане на отзивчив дизайн, който работи добре на настолни компютри, таблети и мобилни устройства.

LINQ е технология, която позволява писане на заявки към бази данни, XML файлове и други източници на данни с помощта на синтаксис, близо до SQL, но интегриран в C#. Това улеснява филтрирането, търсенето и анализа на данните от базата.

public interface IScheduledTaskService
{
    void SendAppointmentReminders();

    void GenerateDailyReports();
    void BackupDatabase();

public class ScheduledTaskService : IScheduledTaskService
{
    private readonly IAppointmentService _appointmentService;
    private readonly IMedicineService _medicineService;
    private readonly INotificationService _notificationService;
    
    public async void SendAppointmentReminders()
    {
        // Получаване на приемни часове за утрешния ден
        var tomorrowAppointments = _appointmentService
            .GetAppointmentsByDate(DateTime.Now.AddDays(1));
        
        foreach (var appointment in tomorrowAppointments)
        {

        }
    }

    {

        foreach (var medicine in expiredMedicines)
        {
            // Логиране и уведомяване
            _medicineService.MarkAsExpired(medicine.MedicineID);
        }
    }
}
