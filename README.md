Model – представя данните и бизнес логиката на приложението (животни, медицински записи, ваксинации, приемни часове)
View – отговаря за презентацията на данните в потребителския интерфейс
Controller – управлява взаимодействието между Model и View, обработвайки потребителските заявки
Този подход позволява добра организация на кода, лесна поддръжка и тестване на приложението.

Информация за ветеринарни услуги

Създаване и управление на график – Системата трябва да позволи на персонала да създава и управлява график на приемни часове за различните ветеринари

Управление на конфликти и отмени – Системата трябва да предотврати конфликти на приемни часове и да позволи отмяна или пренареждане на запланувани консултации

Известяване за предстоящи приемни часове – Системата трябва автоматично да изпраща напомняния до собствениците преди запланувания приемен час

Регистрирање инвентара – Системата трябва да съхранява информация за всички лекарства и материали в клиниката (име, доза, количество, дата на изтичане)

Отчитане на разход – Системата трябва да позволи регистриране на всяко използвано лекарство или материал при лечение на животното

Уведомления за ниски залихи – Системата трябва да алармира при достигане на минимално количество от дадено лекарство

Управление на дати на изтичане – Системата трябва да проследява датите на изтичане и да оповеща персонала за скоро изтичащите лекарства

Управление на преводи – Системата трябва да позволи регистриране на различни услуги (консултация, операция, диагностика) и техните цени

        await _appointmentRepository.SaveChangesAsync();
        return appointment.Id;
    }
    
    public async Task<bool> CancelAppointmentAsync(int id)
    {
        var appointment = await _appointmentRepository.GetByIdAsync(id);
        if (appointment == null) return false;
        
        appointment.Status = AppointmentStatus.Cancelled;
        await _appointmentRepository.UpdateAsync(appointment);
        await _appointmentRepository.SaveChangesAsync();
        return true;
    }
}

    {
        // Получаване на всички предстоящи приемни часове
        var appointments = await _appointmentService.GetAllAppointmentsAsync();
        var tomorrowAppointments = appointments
            .Where(a => a.AppointmentDate.Date == DateTime.Now.AddDays(1).Date &&
                   a.Status == AppointmentStatus.Scheduled);

            _notificationService.SendAppointmentReminder(appointment);
        }
    }
